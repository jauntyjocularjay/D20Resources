# Data Archive (PL 7)

A data archive is a microcomputer implanted in the recipient’s skull. It contains skill-related information stored on a series of biological data chips, or biochips.

## Benefit

The recipient treats all skills as class skills. Furthermore, the recipient gains access to an information database that grants a +4 equipment bonus on all Knowledge checks.

- **Type:** Internal
- **Hardness/Hit Points:** —/2
- **Base Purchase DC:** 27
- **Restriction:** None
